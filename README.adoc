= Simple ToDo App with VueJS and Spring Boot

== VueJS UI

. Create the Project and Insatll Axios
+
[source,shell]
----
vue init webpack todo-ui
cd todo-ui
npm install axios
----

. Add the Bootstrap CSS to the `index.html` file.
+
[source,html]
----
include::todo-ui/index.html[]
----
